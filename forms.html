Form buttons

<input type="submit" value="form input submit" class="sc-btn sc-btn-filled" />
<input type="button" value="form input button" class="sc-btn sc-btn-ghost" />
<button type="button" class="sc-btn sc-btn-shadow">Clickable Button</button>

Disabled buttons can be added as a class name (class="sc-btn-disabled"), or with the disabled attribute ("disabled" or "disabled="disabled").

<input type="submit" 
       value="form input submit" 
       class="sc-btn sc-btn-filled sc-btn-disabled" />
<input type="button"
       value="form input button" 
       class="sc-btn sc-btn-ghost" 
       disabled />
<button type="button" 
        class="sc-btn sc-btn-shadow" 
        disabled="disabled">Clickable Button</button>
